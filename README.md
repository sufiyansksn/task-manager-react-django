## Features

- User registration & login (JWT authentication)
- Create, edit & delete tasks
- Mark tasks as completed
- Search tasks
- Protected routes
- Responsive UI

## Tech Stack

| Frontend | Backend |
|---|---|
| React.js | Django |
| React Router | PostgreSQL |
| JavaScript | Django ORM |
| CSS | Simple JWT |

## Run Locally

**Backend**
```bash
cd Backend
python manage.py runserver
```

**Frontend**
```bash
cd frontend
npm install
npm run dev
```
