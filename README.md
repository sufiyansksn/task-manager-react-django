# Task Manager

A full-stack Task Manager web application built using React, Django, PostgreSQL, and JWT Authentication.

## Features

- User Registration
- User Login
- JWT Authentication
- Create Tasks
- Edit Tasks
- Delete Tasks
- Mark Tasks as Completed
- Search Tasks
- Protected Routes
- Responsive UI

## Tech Stack

### Frontend
- React.js
- React Router
- JavaScript
- CSS

### Backend
- Django
- PostgreSQL
- Django ORM
- Simple JWT

## Project Structure

```
task-manager/
│
├── Backend/
│
└── frontend/
```

## Installation

### Backend

```bash
cd Backend
python3 -m venv env
source env/bin/activate
python manage.py runserver
```

### Frontend

```bash
cd frontend
npm install
npm run dev
```
